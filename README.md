## [сlient](https://github.com/denis-pv/zroip/tree/main/client) 

Клиент для zroip server

запускаем main.py

- скрипт ищет микрофон по умолчанию в системе и берет его.
- при запуске голосовой передачи кодирует звук кодеком G711 и передает на сервер пока удержана комбинация или кнопка PTT
- в директории есть два файла main.py - работает без использования COM порта как вшеняя тангента, 
- maincom.py - ищет в системе COM порт с ардуино, подключается к нему и ждет комманды, при получении комманды запускает и останавливает передачу голоса.

## [PTT](https://github.com/denis-pv/zroip/tree/main/ptt)

Рабочий пример PTT-устройства для ардуино.

### Как это работает

- Ардуино подключено к ПК как usb-com, к ней подключена кнопка тангенты на ногу D9.
- Кнопка замыкает пин на землю.
- Ардуино через usb COM отправляет комманду что кнопка PTT нажата. (XML формат)
- скрипт, maincom.py подключается к этому COM порту, и видит комманду, встает на передачу голоса.
- Когда на ардуино кнопка отпущена, она отсылает комманду о том что кнопка PTT отжата.
- Скрипт завершает передачу голоса на сервер.
